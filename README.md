# 使用 Ant-design-vue



## First
全局安装 vue-cli
```
 yarn global add @vue/cli
```

```
vue create antd-demo
or 
vue create . 
//在当前目录创建
```
## Second

如果使用了第一个命令创建 vue 项目

```
cd my-app
vue add ant-design
```
如果使用第二个命令创建 vue 项目
```
vue add ant-design
```
## Finally
运行 项目
```
 npm run serve
```
